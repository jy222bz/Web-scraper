# A01 Web scraper
## Installation and Configurations:
1. ### Clone the repo:

~~~
clone https://github.com/jy222bz/Web-scraper.git
~~~

2. ### Install the dependencies:
~~~
npm install
~~~

3. ### Run the application:
~~~
npm start 'URL'
~~~
4. ### To run the eslint, run one of the following commands:

~~~
npm test
~~~
~~~
npm run lint
~~~
